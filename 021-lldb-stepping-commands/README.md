## Video Overview

- Go over the basics stepping commands in LLDB for debugging
  - Source level debugging (n, s)
  - Instruction level debugging (ni, si)

## LLDB Commands

```bash
# source-level debugging
n  # next LOC
s  # step into fn

# instruction-level debugging
ni # next assembly LOC
si # step into fn/instruction
```
